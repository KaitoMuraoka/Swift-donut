## build


1. Please `git clone` this repository
``` shell
$ git clone git@github.com:TonTonbow/Swift-donuts.git
```

2. Go to swift-donuts and run `swift build`
```shell
$ cd swift-donuts
$ swift build
```

3. Run `.build/debug/dounts` on the terminal
```shell
$ .build/debug/dounts.
```
